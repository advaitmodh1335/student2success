Software Developer Intern

Tech Corp · Remote · Internship · $25/hr · Deadline: Jan 1, 2026

Description

Join our Developer Experience group to improve how engineers ship code every day. In this role you’ll help design and build internal tools that streamline workflows, reduce toil, and give teams visibility into releases and quality. You’ll collaborate with mentors, product managers, and designers to deliver features incrementally and safely.

Your projects may include enhancing a React component library, adding new API endpoints, instrumenting services for metrics and traces, and building small CLI utilities that automate repetitive tasks. We value thoughtful engineering, clear documentation, and practical testing.

Responsibilities

  • Implement front-end features with React/TypeScript and accessible UI patterns.
  • Build and document JSON/REST endpoints in a simple service architecture.
  • Write automated tests (unit/integration) and keep CI green.
  • Add observability to new code paths (logs, metrics, traces, alerts).
  • Participate in code reviews; give and receive actionable feedback.
  • Collaborate with design on component behavior, states, and edge cases.
  • Contribute to docs, ADRs, and runbooks for easy handoffs.
  • Demo features regularly; iterate quickly based on feedback.

Basic qualifications

  • Foundational web skills: HTML, CSS, JavaScript.
  • Experience with React (or similar) from coursework or projects.
  • Comfort with Git, branches, and pull requests.
  • Understanding of HTTP, REST, and JSON.
  • Strong communication and eagerness to learn.

Preferred qualifications

  • TypeScript and React Hooks familiarity.
  • Some backend experience (Node/Python/Go/Java) and basic SQL.
  • Exposure to Docker and local containerized dev.
  • Awareness of accessibility standards and testing tools.
  • Interest in developer tooling and platform reliability.

Intern experience

  • Dedicated mentor, weekly learning sessions, and project milestones.
  • Opportunities to present your work to engineering and product stakeholders.
  • Potential return-offer pathway based on performance and business needs.

If you enjoy solving real problems for your fellow engineers and want to learn how modern teams ship reliable software, we’d love to meet you.
