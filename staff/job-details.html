Software Developer Intern

Tech Corp · Remote · $25/hr · Submitted 2025-10-20

Description

Join our Developer Experience group to improve how engineers ship code every day. In this role you’ll help design and build internal tools that streamline workflows, reduce toil, and give teams visibility into releases and quality. You’ll collaborate with mentors, product managers, and designers to deliver features incrementally and safely.

Your projects may include enhancing a React component library, adding new API endpoints, instrumenting services for metrics and traces, and building small CLI utilities that automate repetitive tasks. We value thoughtful engineering, clear documentation, and practical testing. Expect code reviews, pairing sessions, and opportunities to demo features and iterate quickly on feedback.

Requirements

  • Proficiency with HTML/CSS/JavaScript (React/TypeScript a plus)
  • Git fundamentals and familiarity with GitHub or GitLab
  • Good communication, curiosity, and willingness to learn
  • Comfort writing unit/integration tests and reading CI results
